# Assassin
Distributed data assistant.

## Structure
- [core](./core) Backend services built on TDN.
- [app](./app) Frontend user interface built on Flutter.


## Built-in Dapps
1. **Yu**: distributed, security and privacy IM.
2. **Sync**: Safely transfer files between devices.
